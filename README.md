# vue base

自定义脚手架（包含 eslint、prettier 配置，vue-router、vuex，sass，flexible & px2rem，以及 vue 的一些基本配置）

## Project setup

```
npm install
```

### Compiles and hot-reloads for development

```
npm run serve
```

### Compiles and minifies for production

```
npm run build
```

### Lints and fixes files

```
npm run lint
```
